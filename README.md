# Bizkhata

Smart Accounting for Growing Businesses.

## Tech Stack
Next.js App Router, TypeScript, Tailwind CSS, Supabase Auth, Supabase PostgreSQL, RLS, Vercel.

## Setup
1. Copy `.env.example` to `.env.local`.
2. Add Supabase URL and anon key.
3. Run `npm install`.
4. Run `npm run dev`.
5. In Supabase SQL Editor, run `supabase/migrations/001_initial_schema.sql`.
6. Deploy through GitHub to Vercel and add the same environment variables.

## MVP Included
- Landing page
- Login/signup
- Dashboard/navigation shell
- MVP module routes
- GST calculation helper
- Double-entry journal helper
- Full initial Supabase schema with RLS policies
